# Absence Management
Models a basic flow of an absence request, built with EF6, ASP.NET WebAPI and AngularJS.

## Technology Used
- **Data**: 
  - [x] [Entity Framework 6](https://github.com/aspnet/EntityFramework6)
- **Domain**: 
  - [x] Date and Time logic [NodaTime](http://nodatime.org/) (*To Be Implemented further; work in progress*)
- **Testing**: 
  - [x] [xUnit](https://github.com/xunit/xunit)
  - [x] [Effort](https://github.com/tamasflamich/effort) (mocking DbContext for EF6)
- **Frontend**:
  - [ ] ASP.Net WebAPI 5, ASP.NET MVC 5 (*To Be Implemented further; work in progress*)
  - [ ] AngularJS 1.6.* (*To Be Implemented further; work in progress*)
  - [ ] Material Design w/ Bootstrap or other solution (*To Be Implemented further; work in progress*)
